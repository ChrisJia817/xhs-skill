# MEMORY.md

## Shared Workflow Preferences

- `xhs-trend-to-publish` 的主链路默认按 `discover -> score -> rewrite -> markdown -> render -> publish` 执行。
- 不要只看关键词搜索结果；真实样本池默认同时纳入：
  1. 搜索结果
  2. 高热作者主页笔记
  3. 首页/推荐页图文
- 目标是草稿箱时，必须走同 session 的“暂存离开”路径，不要把“点击发布”当成“已保存草稿”。
- 发布异常时，优先修 vendor DOM 适配和兼容层，而不是先推翻主流程。
- 图文包默认不少于 4 张：`cover + 至少 2 张正文卡 + 1 张收束卡`。
- 文案要像真人写的，不要写成平台分析报告。
- 正文结尾默认保留话题标签。
- 默认先草稿或私密，公开发布必须显式指定。
