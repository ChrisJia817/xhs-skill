# 评分规则（MVP）

总分 = 热度 + 新鲜度 + 改写潜力 + 账号匹配 - 风险惩罚

建议区间：
- 热度：0-40
- 新鲜度：0-15
- 改写潜力：0-20
- 账号匹配：0-15
- 风险惩罚：0-20

## MVP 简化规则

### 热度
根据点赞/评论/收藏粗略计算：
- 任一指标明显高 → 加分
- 三项都低 → 低分

### 新鲜度
- 近 7 天：高分
- 近 30 天：中分
- 半年内：低到中分

### 改写潜力
满足越多越高：
- 标题明确
- 主题集中
- 有可提炼观点
- 有评论争议点
- 适合图文拆页

### 账号匹配
由 persona/关键词映射决定。

### 风险惩罚
以下情况提高惩罚：
- 夸大收益
- 违规导流
- 敏感词高风险
- 严重同质化
- 低质量搬运痕迹明显
